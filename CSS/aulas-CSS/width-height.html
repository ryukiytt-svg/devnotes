Width e Height

Essas propriedades definem o tamanho da "caixa" do seu elemento.


Como usar

  • Width: Define a largura (ex: width: 300px; ou width: 50%;).
  • Height: Define a altura (ex: height: 100px;).


🛠️ Teste Agora:

  1. No CSS, defina um fundo para o seu h1 para ver o tamanho dele.
  2. Adicione: width: 200px; height: 200px;.
  3. Note que o texto pode "transbordar" se a caixa for muito pequena para ele.
