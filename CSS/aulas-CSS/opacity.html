Propriedade opacity

A propriedade opacity define o nível de transparência de um elemento inteiro (incluindo o texto dentro dele).


Valores

  • 1: Totalmente visível (padrão).
  • 0.5: 50% de transparência.
  • 0: Totalmente invisível (mas ainda ocupa espaço na tela).


🛠️ Teste Agora:

  1. Crie um h1 com fundo colorido.
  2. No CSS, adicione: opacity: 0.3;.
  3. Veja como tudo fica "fantasmagórico".
