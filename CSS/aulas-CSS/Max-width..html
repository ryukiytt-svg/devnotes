Max-width

Diferente do width fixo, o max-width define um limite máximo, mas permite que o elemento diminua se a tela for menor que ele.


Por que usar?

É essencial para sites que funcionam no celular. Se você usa width: 1000px, o site terá barra de rolagem no celular. Com max-width: 1000px; width: 100%;, ele se adapta.



🛠️ Teste Agora:

  1. No CSS, crie uma div com background: red; height: 100px;.
  2. Coloque width: 800px; e veja que ela fica gigante.
  3. Troque para max-width: 800px;. Note que ela respeita o limite máximo.
