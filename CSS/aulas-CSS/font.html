Tamanho e Peso da Fonte

Para controlar o impacto do seu texto, usamos o tamanho e a espessura.


Como usar

  • font-size: Define o tamanho (ex: 24px, 2rem).
  • font-weight: Define se a letra é fina ou grossa (ex: bold ou números como 700).


🛠️ Teste Agora:

  1. Mude o tamanho do seu texto: font-size: 50px;.
  2. Deixe-o bem grosso: font-weight: 900;.
