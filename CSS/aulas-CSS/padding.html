Propriedade padding

O padding é o espaço interno da caixa. Ele cria uma distância entre o texto e a borda.


Como usar

Você pode definir para todos os lados ou individualmente:

  • padding: 20px; (Todos os lados)
  • padding-top: 10px; (Apenas topo)


🛠️ Teste Agora:

  1. Adicione uma borda ao seu h1 para conseguir enxergar o espaço.
  2. No CSS, coloque: padding: 30px;.
  3. Note como o "balão" do título cresce para dentro!
