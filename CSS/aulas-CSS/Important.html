O uso do !important

O !important é uma "marreta" que quebra todas as regras de especificidade. Ele força um estilo a ser aplicado, não importa o quê.


Quando usar?

Dica de ouro: Use o menos possível! Ele dificulta a manutenção do código. É usado apenas em casos extremos onde você não consegue sobrescrever um estilo de outra forma.



🛠️ Teste Agora:

  1. No CSS, coloque:
    h1 { color: blue !important; }
    #unico { color: red; }
  2. Mesmo o ID sendo mais forte, o !important fará o texto ficar azul.
