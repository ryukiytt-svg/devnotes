CSS Transition: A Suavidade

Sem a transição, as mudanças no CSS são instantâneas e secas. A transition permite que a mudança ocorra de forma gradual e elegante.


Como usar:

Você coloca a transição no estado normal do elemento, e não no :hover.

  • transition-property: O que vai mudar (ex: background-color).
  • transition-duration: Quanto tempo demora (ex: 0.5s).
  • transition-timing-function: O "ritmo" (ex: ease-in-out).


🛠️ Teste Agora:

  1. Passe o mouse sobre o botão no playground. Note que ele muda de cor na hora.
  2. No CSS, dentro de .botao, adicione: transition: 0.5s;.
  3. Passe o mouse de novo. Viu como agora ele "desliza" entre as cores?
  4. Desafio: Adicione transform: scale(1.2); dentro do .botao:hover e veja ele crescer suavemente!
