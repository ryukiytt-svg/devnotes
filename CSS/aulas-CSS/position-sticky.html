Position: Sticky

O sticky (pegajoso) é um híbrido. Ele se comporta como relative até que você role a página e ele atinja uma posição específica, aí ele vira fixed.



🛠️ Teste Agora:

  1. Crie uma lista longa de itens no HTML.
  2. No meio dela, coloque um h2 com: position: sticky; top: 0; background: yellow;.
  3. Ao rolar (scroll), o título "gruda" no topo quando chega lá e só desgruda quando a seção dele acaba.
