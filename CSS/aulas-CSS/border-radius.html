Propriedade border-radius

Essa propriedade serve para arredondar os cantos das bordas ou do fundo de um elemento.


Como usar

Quanto maior o valor, mais redondo o canto fica:


  • border-radius: 10px; (Cantos levemente arredondados)
  • border-radius: 50%; (Transforma um quadrado em um círculo)


🛠️ Teste Agora:

  1. No CSS, adicione ao seu elemento:
    2. background-color: orange; border-radius: 20px; padding: 10px;
  2. Tente mudar para 50px e veja a diferença.
