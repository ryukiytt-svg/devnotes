Display: Block

Elementos "Block" são egoístas: eles ocupam toda a largura disponível (100%) e sempre começam em uma nova linha.


Características

  • Respeitam width, height, margin e padding.
  • Exemplos de tags padrão block: <div>, <h1>, <p>.


🛠️ Teste Agora:

  1. No HTML, coloque duas <div> com cores de fundo diferentes.
  2. Note que, mesmo sendo pequenas, uma fica embaixo da outra.
