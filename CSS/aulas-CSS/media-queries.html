Media Queries

As Media Queries permitem que o seu site mude de estilo dependendo do tamanho da tela (celular, tablet ou PC).


  • @media: A regra que inicia a verificação da tela.
  • max-width: Aplica o estilo se a tela for "menor que" o valor definido.


🛠️ Teste Agora:

  1. Clique em Executar e veja que o fundo é cinza.
  2. O código CSS tem uma regra que diz: "Se a tela for menor que 600px, mude o fundo para azul".
  3. Como o nosso iframe geralmente simula uma tela menor, ele deve ficar azul!
