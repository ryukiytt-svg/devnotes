Gradiente Linear

Em vez de uma cor sólida, o gradiente mistura duas ou mais cores em uma direção.


Como usar

Usamos dentro da propriedade background ou background-image:

background: linear-gradient(to right, red, orange);


🛠️ Teste Agora:

  1. No CSS do seu body ou de uma div, coloque:
    2. background: linear-gradient(45deg, #4158D0, #C850C0, #FFCC70);
  2. Clique em Executar e veja o resultado.
