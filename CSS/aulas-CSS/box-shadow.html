Box Shadow

A box-shadow adiciona profundidade ao layout, criando sombras atrás dos elementos (como cards e botões).


Como usar

A ordem dos valores é: deslocamento-x, deslocamento-y, desfoque e cor.

box-shadow: 10px 10px 20px rgba(0,0,0,0.5);


🛠️ Teste Agora:

  1. No CSS, crie um quadrado: width: 100px; height: 100px; background: white;.
  2. Adicione: box-shadow: 0 4px 8px rgba(0,0,0,0.2);.
  3. Veja como ele parece "flutuar" sobre o fundo.
