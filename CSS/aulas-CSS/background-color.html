Propriedade background-color

A propriedade background-color define a cor de fundo de um elemento, como uma div, um p ou a página inteira (body).


Como usar

Para mudar o fundo, escrevemos background-color: seguido da cor escolhida e finalizamos com ;.


Métodos de Cores

Assim como na propriedade color, você pode usar diversos formatos:


  • Nomes: background-color: yellow; (Simples para testes rápidos).
  • Hexadecimal: background-color: #000000; (Preto).
  • RGB/RGBA: background-color: rgba(0, 0, 255, 0.5); (Azul com 50% de transparência).


🛠️ Teste Agora:

Vamos mudar o fundo da página inteira no seu playground:

  1. No campo CSS, adicione este código abaixo do que já existe:
    2. body { background-color: lightgrey; }
  2. Clique em Executar.
  3. Tente trocar lightgrey por #121212 para um "Modo Escuro".
