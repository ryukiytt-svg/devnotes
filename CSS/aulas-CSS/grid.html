CSS Grid: Layout em Duas Dimensões

Diferente do Flexbox (que foca em uma linha ou coluna), o Grid trabalha com linhas E colunas ao mesmo tempo. É como uma planilha de Excel poderosa.


Conceitos Chave:

  • grid-template-columns: Define quantas colunas você quer e qual o tamanho delas.
  • grid-template-rows: Define a altura das linhas.
  • fr (Fractional Unit): Uma unidade especial do Grid que ocupa "uma parte" do espaço disponível.


🛠️ Desafio de Arquitetura:

  1. Veja que os itens estão um embaixo do outro. Vamos criar colunas!
  2. No CSS, no .grade, adicione: grid-template-columns: 1fr 1fr;.
  3. Isso criou duas colunas iguais. Agora mude para 1fr 2fr; (a segunda coluna será o dobro da primeira).
  4. Adicione um gap: 20px; para dar respiro entre as células.
