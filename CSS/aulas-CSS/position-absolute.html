Position: Absolute

O absolute é onde a mágica (e a confusão) acontece. Ele remove o elemento do fluxo da página. É como se ele ganhasse "asas" e flutuasse sobre os outros.


A Regra de Ouro

Um elemento Absolute sempre procura o pai mais próximo que tenha position: relative. Se não achar nenhum, ele se posiciona em relação à tela inteira.



🛠️ Teste Agora:

  1. Crie uma div grande (o pai) com position: relative; height: 200px;.
  2. Dentro dela, crie uma div pequena (o filho) com position: absolute; bottom: 0; right: 0;.
  3. Veja como o filho "gruda" no canto inferior direito do pai!
