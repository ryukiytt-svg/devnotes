Estilizando Imagens

Podemos controlar o tamanho das imagens e até aplicar filtros visuais direto no CSS.


  • object-fit: Faz a imagem preencher o espaço sem ficar esticada.
  • filter: Adiciona efeitos como grayscale (cinza) ou blur (desfoque).


🛠️ Teste Agora:

  1. No CSS, tente mudar o grayscale(1) para sepia(1).
  2. Mude o border-radius para 50% para deixar a imagem redonda.
