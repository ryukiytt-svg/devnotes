Especificidade

Quando dois seletores tentam aplicar cores diferentes ao mesmo elemento, o navegador precisa decidir quem vence. Isso é a especificidade.


A Regra de Ouro (Quem ganha?)

Existe uma hierarquia de "peso". O mais pesado sempre vence:


  1. Style Inline (O mais forte de todos)
  2. ID (#)
  3. Classe (.)
  4. Tag (h1, p) (O mais fraco)


🛠️ Teste Agora:

Vamos ver a briga de forças no seu playground:

  1. No CSS, coloque:
    h1 { color: red; } (Tag)
    .titulo { color: green; } (Classe)
  2. No HTML use: <h1 class="titulo">Quem vence?</h1>
  3. O texto ficará Verde, pois a Classe é mais específica que a Tag!
