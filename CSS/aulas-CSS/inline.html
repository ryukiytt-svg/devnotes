Display: Inline

Elementos "Inline" são sociáveis: eles ocupam apenas o espaço necessário para o seu conteúdo e permitem que outros elementos fiquem ao seu lado.


Características

  • Não aceitam width nem height.
  • Margens e paddings laterais funcionam, mas os verticais não empurram outros elementos.
  • Exemplos: <span>, <a>, <strong>.


🛠️ Teste Agora:

  1. No HTML, crie três <span>.
  2. No CSS, tente dar um width: 500px; para eles.
  3. Veja que nada acontece! O tamanho deles é travado no texto.
