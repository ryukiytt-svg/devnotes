Dominando Flexbox

O Flexbox transforma um elemento em um "Pai Elástico" (Container). Quando você ativa o display: flex, todos os filhos diretos viram "Itens Flexíveis".


1. A Batalha dos Eixos

Para dominar o Flexbox, você precisa entender que ele trabalha com dois caminhos:


  • Main Axis (Eixo Principal): Por padrão, é a linha (horizontal).

  • Cross Axis (Eixo Cruzado): É a linha perpendicular (vertical).

2. Propriedades do Container (O Pai)

  • flex-direction: Define se os itens ficam em linha (row) ou pilha (column).

  • justify-content: Alinha os itens no Eixo Principal (Horizontal).
    Valores: center, flex-start, flex-end, space-between (espalha tudo), space-around.

  • align-items: Alinha os itens no Eixo Cruzado (Vertical).
    Valores: center, stretch (estica), flex-start.


🛠️ Desafio de Mestre:

  1. Clique em Executar. Veja que os itens estão espremidos na esquerda.
  2. No CSS, no .container, adicione justify-content: space-around;.
  3. Agora, adicione align-items: center;.
  4. O Grande Truque: Mude o flex-direction para column e veja como o justify-content agora controla a vertical!
