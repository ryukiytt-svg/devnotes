Alinhamento de Texto

A propriedade text-align controla como o texto se posiciona horizontalmente dentro da caixa.


Valores comuns

  • left: Alinha à esquerda (padrão).
  • center: Centraliza o texto.
  • right: Alinha à direita.
  • justify: Estica o texto para ocupar toda a largura.


🛠️ Teste Agora:

  1. No CSS do seu h1, coloque: text-align: center;.
  2. Aumente a width para 100% e veja o título ir para o meio da tela.

Estilização de Texto

Podemos transformar o texto e adicionar ou remover decorações (como sublinhados).


Propriedades

  • text-decoration: Usado para colocar underline (sublinhado) ou remover o sublinhado de links com none.
  • text-transform: Transforma o texto em uppercase (MAIÚSCULAS) ou lowercase (minúsculas).


🛠️ Teste Agora:

  1. No seu CSS, adicione:
    2. text-transform: uppercase; text-decoration: underline;
  2. Clique em Executar.

Exemplos de Valores


none: Remove qualquer decoração existente.

underline: Desenha uma linha abaixo do texto (padrão para links).

overline: Desenha uma linha acima do texto.

line-through: Desenha uma linha no meio do texto (riscado).

blink: Faz o texto piscar (descontinuado, não suportado por navegadores modernos).
