Position: Relative

O relative é o posicionamento mais "seguro". Ele mantém o elemento no fluxo natural da página, mas permite que você o mova um pouco usando top, bottom, left ou right.


O que acontece?

  • O elemento não sai do lugar original dele para os outros vizinhos.
  • Ele apenas ganha um "offset" (deslocamento).
  • Ele serve de "âncora" para elementos absolute (veremos na próxima aula).


🛠️ Teste Agora:

  1. Crie três div coloridas no HTML.
  2. No CSS da segunda div, coloque:
    position: relative;
    top: 20px; left: 30px;
  3. Note que ela se move, mas as outras divs continuam no mesmo lugar, como se a segunda ainda estivesse lá.
