Propriedade color

A propriedade color define a cor do texto de um elemento.


Como usar

Para aplicar as cores, primeiro digitamos color:, depois colocamos a cor (ex: red) e finalizamos com ;.


Métodos para aplicar cores

Existem diversas formas de definir uma cor no CSS. As principais são:


  • 1. Nome da cor: Digitando o nome em inglês, como red, green ou blue. Nota: Existem apenas 140 cores predeterminadas, por isso é um método limitado.
  • 2. Códigos de cores: Para maior precisão, usamos sistemas como: hex, rgb, rgba, hsl, hsla, hwb, lab, lch, oklab e oklch.

  1. Vá no campo CSS.
  2. Apague o red.
  3. Digite hotpink.
