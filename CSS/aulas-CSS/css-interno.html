CSS Interno (Embedded)

O CSS Interno é definido dentro do próprio documento HTML, mas separado das tags, usando a tag <style>.


Como usar

Geralmente ele fica dentro do <head> do HTML. Você usa seletores para dizer qual elemento quer mudar.



🛠️ Teste Agora:

  1. No campo HTML, adicione isso acima do seu H1:
    2. <style> h1 { background-color: yellow; } </style>
  2. Clique em Executar.
