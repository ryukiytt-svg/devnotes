Seletores CSS

Seletores são usados para "alvo" nos elementos HTML que queremos estilizar.


Tipos de Seletores

  • Seletor de Tag: Afeta todas as tags daquele tipo (ex: h1, p, div).
  • Seletor de Classe (.): Usado para estilizar vários elementos diferentes. No HTML usamos class="nome" e no CSS .nome.
  • Seletor de ID (#): Usado para um elemento único. No HTML usamos id="topo" e no CSS #topo.


🛠️ Teste Agora:

  1. No HTML, coloque: <h1 class="titulo" id="unico">Teste de Seletores</h1>
  2. No CSS, tente usar h1 { color: red; }, depois .titulo { color: blue; } e veja qual vence.
  3. Agora adicione #unico { color: green; } e veja qual vence.
