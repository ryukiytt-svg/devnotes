Position: Fixed

O fixed ignora todo mundo e se prende à **janela do navegador** (viewport). Ele não liga para quem é o pai dele.


Onde usamos?

Sabe aquele botão de "Voltar ao topo" ou um menu que não some quando você rola a página? É o Fixed!



🛠️ Teste Agora:

  1. No CSS, coloque no seu h1:
    2. position: fixed; top: 0; left: 0; width: 100%; background: black; color: white;
  2. Agora tente imaginar a página com muito texto: o título nunca sairia do topo da sua visão.
